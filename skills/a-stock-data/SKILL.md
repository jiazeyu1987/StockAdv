# A股日线数据获取技能

## 任务目标
- 本技能用于获取A股市场（上海证券交易所、深圳证券交易所）的日线行情数据
- 支持获取指数数据（如上证指数、深证成指、创业板指等）
- 支持获取个股数据（通过股票代码或名称查询）
- 数据包含：开盘价、最高价、最低价、收盘价、涨跌值、涨跌幅、成交量、成交额

## 能力范围
1. **指数数据获取**：上证指数、深证成指、创业板指、科创50、沪深300等主流指数
2. **个股数据获取**：通过6位数字代码或股票名称查询
3. **历史数据查询**：支持获取指定时间范围的日线数据
4. **实时行情获取**：获取最新交易日数据

## 触发条件
- 当用户提到"A股数据"、"股票日线"、"历史行情"、"获取股票数据"时使用
- 当用户需要"查询指数"、"个股数据"、"K线数据"时使用
- 当用户输入股票代码（如600519、000001）或股票名称（如贵州茅台、平安银行）时使用

## 操作步骤

### 1. 识别股票代码
- 用户输入可能是：
  - 6位数字代码（如600519、000001、300750）
  - 股票名称（如"贵州茅台"、"宁德时代"）
  - 指数名称（如"上证指数"、"沪深300"）
- 如果是名称，需要先转换为对应的6位代码

### 2. 确定数据范围
- 获取历史数据：指定开始日期和结束日期
- 获取最新数据：获取最近N个交易日数据
- 默认获取最近30个交易日的数据

### 3. 调用数据获取脚本
- 当用户需要获取A股日线数据时，**必须调用** [scripts/fetch-stock-data.js](scripts/fetch-stock-data.js)
- 脚本参数：
  - `code`: 股票代码（6位数字）
  - `startDate`: 开始日期（YYYY-MM-DD格式，可选）
  - `endDate`: 结束日期（YYYY-MM-DD格式，可选）
  - `days`: 获取最近N天数据（默认30天，可选）

### 4. 返回数据格式
数据返回JSON格式，包含以下字段：
```json
{
  "code": "600519",
  "name": "贵州茅台",
  "data": [
    {
      "date": "2024-01-15",
      "open": 1688.00,
      "high": 1705.00,
      "low": 1675.00,
      "close": 1698.00,
      "change": 12.00,
      "changePercent": 0.71,
      "volume": 28500,
      "amount": 48330000
    }
  ]
}
```

### 个股代码规则
- 沪市A股：600/601/603/605开头（如600519贵州茅台）
- 深市主板：000/001开头（如000001平安银行）
- 中小板：002开头
- 创业板：300开头（如300750宁德时代）
- 科创板：688开头
- 北交所：8开头

- **脚本调用规则**：获取A股数据涉及外部API调用和数据解析，**必须调用** scripts/ 中的脚本，不要尝试自行构造数据
- **数据延迟**：日线数据通常在交易日收盘后更新，实时数据可能有15分钟延迟
- **代码格式**：沪市股票代码需添加前缀（如sh600519），深市需添加前缀（如sz000001），脚本会自动处理
