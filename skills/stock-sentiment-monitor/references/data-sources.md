# 股票舆情数据来源

## 概述
本文档列出可用于抓取股票舆情信息的公开数据源。

## 数据源列表

### 1. 新浪财经
- **网址**：https://finance.sina.com.cn
- **数据类型**：新闻、公告、研报
- **访问方式**：网页抓取
- **更新频率**：实时
- **注意事项**：需遵守 robots.txt

### 2. 东方财富
- **网址**：https://finance.eastmoney.com
- **数据类型**：新闻、公告、研报、股吧讨论
- **访问方式**：网页抓取 / API
- **更新频率**：实时
- **注意事项**：部分数据需登录

### 3. 同花顺
- **网址**：https://www.10jqka.com.cn
- **数据类型**：新闻、公告、研报
- **访问方式**：网页抓取
- **更新频率**：实时

### 4. 巨潮资讯网
- **网址**：http://www.cninfo.com.cn
- **数据类型**：官方公告、定期报告
- **访问方式**：API / 网页抓取
- **更新频率**：实时
- **特点**：证监会指定信息披露网站

### 5. 上交所/深交所公告
- **上交所**：http://www.sse.com.cn
- **深交所**：http://www.szse.cn
- **数据类型**：官方公告、监管信息
- **访问方式**：API
- **特点**：最权威的信息来源

## 抓取策略

### 推荐优先级
1. 巨潮资讯网（公告类）
2. 新浪财经（新闻类）
3. 东方财富（综合类）

### 抓取频率建议
- 实时监控：每 15 分钟
- 日常监控：每日一次
- 批量分析：每周一次

### 数据存储
- 原始数据：JSON 格式
- 分析结果：Excel 报告
- 历史数据：数据库存储（可选）
