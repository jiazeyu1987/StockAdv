# 宏观经济指标参考手册

## 一、利率与流动性指标

### 1.1 政策利率
| 指标 | 定义 | 影响方向 | 数据来源 |
|-----|------|---------|---------|
| 联邦基金利率 | 美国银行间隔夜拆借利率 | 全球资金成本基准 | FRED |
| 央行基准利率 | 各国央行政策利率 | 国内资金成本 | 央行官网 |
| 存款准备金率 | 商业银行准备金比例 | 货币乘数 | 央行官网 |

### 1.2 市场利率
| 指标 | 定义 | 分析要点 |
|-----|------|---------|
| 国债收益率（10Y） | 无风险利率基准 | 期限利差、实际利率 |
| 信用利差（BBB-国债） | 信用风险溢价 | 风险偏好指标 |
| LIBOR/SOFR | 银行间拆借利率 | 流动性状况 |

### 1.3 流动性指标
| 指标 | 计算公式 | 解读 |
|-----|---------|------|
| 实际利率 | 名义利率 - 通胀率 | 负值刺激风险资产 |
| 期限利差 | 10Y收益率 - 2Y收益率 | 倒挂预示衰退 |
| 信用脉冲 | 信贷增量/GDP | 领先经济6-9个月 |

## 二、通胀与增长指标

### 2.1 通胀指标
| 指标 | 覆盖范围 | 发布频率 | 重要性 |
|-----|---------|---------|-------|
| CPI | 消费者价格 | 月度 | ★★★★★ |
| PPI | 生产者价格 | 月度 | ★★★★ |
| PCE | 个人消费支出价格 | 月度 | ★★★★★（美联储目标） |
| 核心通胀 | 剔除食品能源 | 月度 | ★★★★★ |

### 2.2 增长指标
| 指标 | 定义 | 领先/同步/滞后 |
|-----|------|---------------|
| PMI | 采购经理人指数 | 领先 |
| GDP | 国内生产总值 | 同步 |
| 工业增加值 | 工业产出变化 | 同步 |
| 零售销售 | 消费支出 | 同步 |
| 失业率 | 劳动力市场状况 | 滞后 |

### 2.3 产出缺口
- **定义**：实际产出与潜在产出的差额
- **计算**：(实际GDP - 潜在GDP) / 潜在GDP × 100%
- **意义**：正值表示经济过热，负值表示产能过剩

## 三、汇率与跨境资本

### 3.1 汇率指标
| 指标 | 说明 | 影响 |
|-----|------|------|
| 美元指数（DXY） | 美元兑一篮子货币 | 全球流动性风向标 |

| 外汇储备变动 | 央行 | 资本进出压力 |
| 北向/南向资金 | 交易所 | A股外资流向 |
| 国际收支 | 外汇管理局 | 经常账户+资本账户 |

## 四、政策分析框架

### 4.1 货币政策
| 维度 | 分析要点 |
|-----|---------|
| 方向 | 宽松/紧缩/中性 |
| 力度 | 降息幅度/降准幅度/QE规模 |
| 持续性 | 政策空间、制约因素 |
| 传导 | 政策利率→市场利率→实体经济 |

### 4.2 财政政策
| 维度 | 分析要点 |
|-----|---------|
| 赤字率 | 财政扩张力度 |
| 专项债 | 基建投资规模 |
| 减税降费 | 企业盈利影响 |
| 支出结构 | 消费/投资/转移支付 |

### 4.3 产业政策
| 维度 | 分析要点 |
|-----|---------|
| 支持方向 | 战略新兴产业/传统升级 |
| 补贴力度 | 直接补贴/税收优惠 |
| 准入门槛 | 行业壁垒变化 |
| 监管趋势 | 放松/收紧 |

## 五、地缘政治与事件分析

### 5.1 冲击传导矩阵

```
能源冲击 → 运价上涨 → 通胀上升 → 利率上行 → 风险偏好下降
    ↓
制裁措施 → 供给缺口 → 替代需求 → 受益行业/受损行业
    ↓
地缘冲突 → 避险需求 → 美元/黄金上涨 → 风险资产下跌
```

### 5.2 事件分类与影响
| 事件类型 | 典型影响 | 受益资产 | 受损资产 |
|---------|---------|---------|---------|
| 能源危机 | 通胀上行 | 能源股、黄金 | 成长股、消费股 |
| 贸易摩擦 | 供应链重构 | 国产替代 | 出口依赖型企业 |
| 金融制裁 | 支付体系割裂 | 数字货币、黄金 | 跨境银行 |
| 战争冲突 | 避险情绪 | 军工、黄金、美元 | 风险资产 |

## 六、库存周期

### 6.1 周期阶段
| 阶段 | 特征 | 资产表现 |
|-----|------|---------|
| 主动补库存 | 需求↑+库存↑ | 周期股、商品 |
| 被动补库存 | 需求↓+库存↑ | 谨慎 |
| 主动去库存 | 需求↓+库存↓ | 防御股 |
| 被动去库存 | 需求↑+库存↓ | 布局周期股 |

### 6.2 领先指标
- 原材料库存PMI
- 产成品库存PMI
- 原材料价格指数
- 新订单指数
