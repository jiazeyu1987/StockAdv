# 风险评估框架

## 一、单一标的风险指标

### 1.1 波动率指标

#### 历史波动率
```
σ = √(Σ(Ri - R̄)² / (n-1)) × √252

其中：
Ri = 日收益率
R̄ = 平均日收益率
n = 交易日数量
252 = 年化交易日
```

**解读**：
- σ < 20%：低风险
- 20% ≤ σ < 40%：中等风险
- σ ≥ 40%：高风险

#### 隐含波动率
- 从期权价格反推的市场预期波动
- VIX指数：标普500隐含波动率

---

### 1.2 风险价值（VaR）

**定义**：在给定置信水平下，特定时间内可能的最大损失。

**计算方法**：

**参数法**：
```
VaR = 投资组合价值 × (μ - z×σ)

其中：
μ = 预期收益率
z = 标准正态分布分位数（95%置信度z=1.65，99%置信度z=2.33）
σ = 波动率
```

**历史模拟法**：
- 使用历史收益率分布
- 取分位数作为VaR估计

**蒙特卡洛模拟**：
- 模拟多种情景下的收益分布
- 计算分位数VaR

---

### 1.3 最大回撤（Max Drawdown）

**公式**：
```
最大回撤 = max[(峰值 - 谷值) / 峰值]
```

**分析要点**：
- 回撤持续时间
- 回撤恢复时间
- 与基准对比

---

### 1.4 下行风险指标

#### 下行标准差
```
下行σ = √(Σmin(Ri - R̄, 0)² / n)
```

#### Sortino比率
```
Sortino = (Rp - Rf) / 下行σ

其中：
Rp = 投资组合收益
Rf = 无风险利率
```

---

## 二、组合风险指标

### 2.1 集中度风险

#### 单一标的权重
```
集中度 = 单一标的市值 / 组合总市值
```

**风控阈值**：
- 单一标的 ≤ 10%（一般股票）
- 单一标的 ≤ 5%（高风险标的）
- 单一行业 ≤ 30%

#### 赫芬达尔指数（HHI）
```
HHI = Σ(权重i)²

解读：
HHI < 0.15：分散
0.15 ≤ HHI < 0.25：适度集中
HHI ≥ 0.25：高度集中
```

---

### 2.2 相关性风险

#### 组合波动率
```
σp = √(ΣΣ wi×wj×σi×σj×ρij)

其中：
wi, wj = 资产i,j的权重
σi, σj = 资产i,j的波动率
ρij = 资产i,j的相关系数
```

#### 分散化效应
```
分散化效应 = 1 - σp / Σ(wi×σi)
```

---

### 2.3 因子暴露

#### 常见风险因子
| 因子 | 度量方式 | 风险来源 |
|-----|---------|---------|
| 市场风险（Beta） | 对市场收益的敏感度 | 系统性风险 |
| 规模因子 | 市值暴露 | 小盘股风险 |
| 价值因子 | 估值暴露 | 价值陷阱 |
| 动量因子 | 趋势暴露 | 趋势反转 |
| 质量因子 | 盈利质量暴露 | 盈利下滑 |

#### Beta计算
```
βi = Cov(Ri, Rm) / Var(Rm)

其中：
Ri = 个股收益
Rm = 市场收益
```

---

## 三、流动性风险

### 3.1 流动性指标

#### 买卖价差
```
价差% = (卖一价 - 买一价) / 中间价 × 100%
```

**阈值**：
- < 0.1%：高流动性
- 0.1% - 0.5%：中等流动性
- > 0.5%：低流动性

#### 去化天数（Days to Liquidate）
```
去化天数 = 持仓量 / 日均成交量
```

**阈值**：
- < 1天：高流动性
- 1-5天：中等流动性
- > 5天：低流动性

---

### 3.2 市场冲击成本

#### Kyle Lambda
```
价格冲击 = λ × 交易量

λ = 市场深度指标
```

#### 滑点估计
```
预期滑点 = 交易量 / 市场深度 × 价差
```

---

## 四、尾部风险

### 4.1 预期亏损（CVaR/ES）

**定义**：超过VaR阈值后的平均损失。

**公式**：
```
CVaR = E[损失 | 损失 > VaR]
```

**优势**：
- 考虑尾部极端情况
- 满足次可加性（组合CVaR ≤ 各资产CVaR之和）

---

### 4.2 偏度与峰度

#### 偏度（Skewness）
```
偏度 = E[(R - μ)³] / σ³

解读：
负偏度：左尾风险大
正偏度：右尾机会大
```

#### 峰度（Kurtosis）
```
峰度 = E[(R - μ)⁴] / σ⁴

解读：
峰度 > 3：厚尾，极端事件概率高
峰度 < 3：薄尾，极端事件概率低
```

## 五、压力测试

### 5.1 历史情景

| 情景 | 时间 | 市场跌幅 | 适用性 |
|-----|------|---------|-------|
| 2008金融危机 | 2008-2009 | -50% | 系统性风险 |
| 2015A股异常波动 | 2015 | -40% | 流动性风险 |
| 2020疫情冲击 | 2020Q1 | -35% | 黑天鹅事件 |
| 2022加息周期 | 2022 | -25% | 利率风险 |

### 5.2 假设情景

| 情景 | 假设条件 | 测试重点 |
|-----|---------|---------|
| 经济衰退 | GDP负增长 | 盈利下滑 |
| 利率急升 | 利率+200bp | 估值压缩 |
| 汇率波动 | 本币贬值20% | 出口/进口影响 |
| 行业冲击 | 政策收紧 | 监管风险 |

## 六、风险限额体系

### 6.1 限额层级

| 层级 | 限额类型 | 监控频率 |
|-----|---------|---------|
| 公司级 | 总敞口、VaR | 日度 |
| 组合级 | 行业集中度、因子暴露 | 日度 |
| 标的级 | 单一标的权重、止损 | 实时 |

### 6.2 常用限额

| 指标 | 软限 | 硬限 |
|-----|------|------|
| 单日VaR（95%） | 组合价值2% | 组合价值3% |
| 最大回撤 | 15% | 20% |
| 单一标的权重 | 8% | 10% |
| 单一行业权重 | 25% | 30% |
| 杠杆倍数 | 1.5x | 2x |

### 6.3 处置曲线

| 触发条件 | 处置措施 |
|---------|---------|
| 触及软限 | 减仓预警、增加对冲 |
| 触及硬限 | 强制减仓、暂停新增 |
| 连续触及 | 策略审查、暂停交易 |
