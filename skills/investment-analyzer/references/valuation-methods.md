# 估值方法参考手册

## 一、相对估值法

### 1.1 市盈率（P/E）

**公式**：
```
P/E = 股价 / 每股收益（EPS）
或
P/E = 市值 / 净利润
```

**适用场景**：
- 盈利稳定、周期性弱的行业
- 消费、医药、科技（成熟期）

**分析要点**：
- 历史分位：当前PE vs 5年/10年区间
- 同业对比：vs 行业平均/龙头
- 增长调整：PEG = PE / 盈利增长率

**注意事项**：
- 亏损企业无法使用
- 一次性收益需调整
- 不同会计政策影响可比性

---

### 1.2 市净率（P/B）

**公式**：
```
P/B = 股价 / 每股净资产
或
P/B = 市值 / 净资产
```

**适用场景**：
- 重资产行业：银行、保险、房地产
- 周期性行业：钢铁、煤炭
- 清算价值评估

**分析要点**：
- ROE-PB关系：合理PB ≈ ROE/要求回报率
- 账面价值质量：无形资产占比
- 破净分析：P/B < 1的原因

---

### 1.3 市销率（P/S）

**公式**：
```
P/S = 市值 / 营业收入
```

**适用场景**：
- 亏损或微利企业
- 高成长早期企业
- 收入质量稳定的行业

**分析要点**：
- 结合毛利率分析
- 收入确认政策
- 增长可持续性

---

### 1.4 EV/EBITDA

**公式**：
```
EV = 市值 + 总负债 - 现金
EBITDA = 息税前利润 + 折旧 + 摊销
EV/EBITDA = 企业价值 / 息税折旧摊销前利润
```

**适用场景**：
- 资本结构差异大的公司对比
- 重资产、高折旧行业
- 并购估值

**优势**：
- 不受资本结构影响
- 不受折旧政策影响
- 国际可比性强

---

## 二、绝对估值法

### 2.1 股利贴现模型（DDM）

**公式**：
```
P = D₁ / (r - g)

其中：
D₁ = 下期预期股利
r = 要求回报率（股权成本）
g = 股利永续增长率
```

**适用场景**：
- 股利政策稳定的公司
- 成熟行业龙头
- 公用事业、REITs

**参数估计**：
- g通常取GDP增速或略高（2-4%）
- r可用CAPM模型估算

---

### 2.2 自由现金流折现（DCF）

**公式**：
```
企业价值 = Σ [FCFₜ / (1+WACC)ᵗ] + 终值 / (1+WACC)ⁿ

股权价值 = 企业价值 - 净负债
```

**自由现金流计算**：
```
FCFF（企业自由现金流）= EBIT×(1-税率) + 折旧摊销 - 资本支出 - 营运资金增加

FCFE（股权自由现金流）= 净利润 + 折旧摊销 - 资本支出 - 营运资金增加 + 净借债
```

**适用场景**：
- 现金流稳定可预测的企业
- 并购估值
- 内在价值评估

**关键参数**：
| 参数 | 估算方法 |
|-----|---------|
| WACC | 加权平均资本成本 |
| 增长率 | 分阶段预测（高增→稳增） |
| 终值 | Gordon增长模型或退出倍数 |
| 预测期 | 通常5-10年 |

---

### 2.3 剩余收益模型

**公式**：
```
股权价值 = 账面价值 + Σ [RIₜ / (1+r)ᵗ]

剩余收益 RI = 净利润 - 股权成本×期初账面价值
            = (ROE - r) × 期初账面价值
```

**适用场景**：
- 会计盈利与价值创造关联强的企业
- 银行、保险公司
- 与EVA（经济增加值）概念一致

---

## 三、分部估值法（SOTP）

### 3.1 方法说明

对多元化经营企业的各业务板块分别估值，加总得到整体价值。

**公式**：
```
总价值 = Σ（各业务板块价值）+ 非经营性资产 - 母公司费用 - 净负债
```

**适用场景**：
- 多元化集团企业
- 业务差异大的公司
- 并购拆分估值

### 3.2 估值方法选择

| 业务类型 | 推荐估值方法 |
|---------|-------------|
| 成熟稳定业务 | PE、EV/EBITDA |
| 高成长业务 | P/S、PEG、DCF |
| 重资产业务 | PB、EV/产能 |
| 金融业务 | PB、PE |
| 亏损业务 | 收入倍数、P/用户 |

---

## 四、特殊估值方法

### 4.1 PEG比率

**公式**：
```
PEG = PE / 盈利增长率（%）
```

**判断标准**：
- PEG < 1：可能被低估
- PEG = 1：合理估值
- PEG > 1：可能被高估

**注意事项**：
- 增长率需用未来预期增长
- 不适用于低增长或负增长企业

---

### 4.2 风险投资估值法

**公式**：
```
投后估值 = 预期退出估值 / (1+要求回报率)ⁿ

要求回报率：早期50%+，成长期30-40%，Pre-IPO 20-30%
```

**适用场景**：
- 未盈利高成长企业
- 风险投资估值
- 一级市场投资

---

## 五、估值方法选择决策树

```
是否有稳定盈利？
├── 是 → 是否有稳定现金流？
│       ├── 是 → DCF（首选）或 PE
│       └── 否 → PE 或 PB
└── 否 → 是否有稳定收入？
        ├── 是 → P/S 或 EV/收入
        └── 否 → 资产法或实物期权

是否为重资产行业？
├── 是 → PB 或 EV/产能
└── 否 → 上述流程

是否为多元化集团？
├── 是 → SOTP
└── 否 → 上述流程
```

## 六、估值区间计算

### 6.1 敏感性分析

**关键变量敏感性**：
| 估值方法 | 关键变量 | 敏感性 |
|---------|---------|-------|
| DCF | WACC、增长率 | 高 |
| PE | 盈利预测 | 中 |
| PB | ROE | 中 |
| PEG | 增长率假设 | 高 |

### 6.2 估值区间表达

**标准格式**：
```
基于[方法1]，目标价[价格1]元（假设[关键假设1]）
基于[方法2]，目标价[价格2]元（假设[关键假设2]）
综合估值区间：[低价]-[高价]元
中值：[中值]元，对应[上涨/下跌]空间[幅度]%
```

## 七、常见估值陷阱

### 7.1 会计操纵识别
- 收入提前确认
- 费用资本化
- 商誉减值风险
- 关联交易影响

### 7.2 周期股估值
- 避免在周期顶点用当期盈利估值
- 使用周期平均盈利或正常化盈利
- 结合PB和EV/产能

### 7.3 成长股估值
- 区分可持续增长与一次性增长
- 警惕过高增长假设
- 考虑竞争加剧对增长的侵蚀
