# 股票分析报告可视化

## 任务目标
- 本 Skill 用于生成专业的股票/投资分析报告可视化页面
- 能力：多空分歧分析、Agent评分展示、财务指标可视化、行业筛选、估值对比
- 触发：当用户需要股票分析报告、投资研究可视化、多空逻辑对比时使用

## 前置准备
- 依赖说明：无特殊依赖，使用原生 HTML/CSS/JS
- 数据准备：用户提供股票分析数据

## 操作步骤
1. 接收用户提供的股票分析数据
2. 生成包含以下模块的可视化报告页面：
   - 多空分歧分析（多方核心逻辑 vs 空方核心风险）
   - 三个Agent评分（基本面Agent、技术面Agent、催化剂Agent）
   - 核心财务指标展示
   - 净利润趋势图
   - 业务板块分析
   - 估值对比卡片
   - 宏观Agent输出
   - 行业筛选Agent评分
3. 输出完整的 HTML 文件

## 页面结构说明

### 1. 多空分歧模块
- 左右两栏布局：左侧多方（绿色主题），右侧空方（红色主题）
- 每个观点包含标题和详细描述
- 底部综合评级区域

### 2. Agent评分模块
- 三个Agent卡片：基本面(F)、技术面(T)、催化剂(C)
- 每个Agent包含：总评分、细分指标进度条、详细分析文字
- 进度条颜色区分不同Agent

### 3. 财务指标模块
- 核心财务数据卡片（营收、净利润、分红等）
- 净利润趋势横向条形图
- 业务板块分析（油轮、其他业务）

### 4. 估值对比模块
- 四列卡片布局：PE、股息率、目标价等

### 5. 宏观与行业筛选
- 宏观Agent输出（经济周期、货币政策、风险偏好）
- 行业筛选Agent评分卡片

## 注意事项
- **附件读取规则**：生成报告前，**必须先读取** [assets/template.html](assets/template.html) 了解页面结构和样式规范
- **脚本调用规则**：处理数据生成报告时，**立即调用** [scripts/generate-report.js](scripts/generate-report.js)
- **数据格式**：输入数据应为 JSON 格式，包含所有必要的分析字段
