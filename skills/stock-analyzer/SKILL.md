# 股票分析技能

## 任务目标
- 本 Skill 用于: 分析指定股票并生成专业研报
- 能力: 消息面分析、技术面分析、走势判断、投资建议
- 触发: 用户提供股票代码或名称，要求分析

## 前置准备
- 依赖说明: 无需额外依赖，使用智能体原生能力
- 数据来源: 通过 web_search 工具搜索股票相关新闻、公告、研报数据
- **数据时效性**: 搜索时必须使用最新日期数据，当前日期为2026年3月23日

## 操作步骤

### 1. 股票识别与信息获取
- 解析用户输入，识别股票代码或名称
- 使用 web_search 搜索股票基本信息（代码、名称、所属行业、主营业务）
- 搜索股票近期新闻和公告
- **搜索关键词格式**: "[股票名称] 股票 [当前年份]年3月" 或 "[股票代码] 股票最新消息"

### 2. 消息面分析
- 搜索并分析近期新闻动态
- **搜索关键词**: "[股票名称] 新闻 2026年3月"、"[股票名称] 公告 最新"
- 分析公司公告和重大事项
- 评估市场情绪和舆论导向
- 分析行业政策和趋势

### 3. 技术面分析
- 搜索股票近期价格走势和成交量数据
- **搜索关键词**: "[股票名称] 技术分析 2026年3月"、"[股票代码] 股价 走势"
- 分析K线形态和关键点位
- 分析均线系统（MA5、MA10、MA20、MA60等）
- **搜索技术指标**: "[股票名称] MACD KDJ RSI 2026"
- 分析常用技术指标（MACD、KDJ、RSI、布林带等）
- 判断趋势方向和支撑阻力位

### 4. 研报生成
- 综合消息面和技术面分析
- 给出投资评级和建议
- 提示风险因素
- 形成结构化研报输出

## 研报输出格式

```
# [股票名称]([股票代码]) 分析研报

## 一、基本信息
- 股票代码: 
- 股票名称: 
- 所属行业: 
- 主营业务: 

## 二、消息面分析
### 2.1 近期新闻
[新闻摘要]
### 2.2 公告解读
[重要公告分析]
### 2.3 行业动态
[行业政策和趋势]

## 三、技术面分析
### 3.1 走势概述
[近期价格走势描述]
### 3.2 技术指标分析
[MACD、KDJ、RSI等指标分析]
### 3.3 趋势判断
[多空方向判断]

## 四、综合研判
### 4.1 投资评级
[买入/持有/卖出建议]
### 4.2 目标价位
[预期目标价]
### 4.3 风险提示
[主要风险因素]

## 五、操作建议
[具体买卖建议]
```

## 注意事项
- 消息面分析需引用真实新闻来源，标注搜索时间
- 技术面分析需基于实际行情数据，避免主观臆测
- 投资评级需客观谨慎，提示投资风险
- 研报内容需结构清晰，数据详实
- 明确告知用户本分析仅供参考，不构成投资建议
