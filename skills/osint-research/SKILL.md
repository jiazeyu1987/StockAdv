# OSINT 开源情报调研技能

## 任务目标

本 Skill 用于执行专业的开源情报（OSINT）调研任务，采用螺旋迭代和种子扩散方法论，系统化地收集和分析互联网公开信息。

**核心能力：**
- 竞争对手企业信息调研（工商信息、融资情况、团队背景）
- 供应链上下游企业关系图谱构建
- 行业趋势和市场格局分析
- 关键人员背景调查
- 公开数据源的自动化采集

**触发场景：**
- 用户要求进行竞品分析或竞争对手调研
- 需要了解某企业的供应链上下游关系
- 进行行业研究或市场格局分析
- 企业背景调查或尽职调查
- 需要收集公开的企业信息或人员信息

## 方法论概述

### 螺旋迭代模型

```
初始种子 → 信息收集 → 分析验证 → 发现新线索 → 扩大搜索 → 循环迭代
```

1. **初始种子**：从已知的关键信息（公司名、人名、产品名）开始
2. **信息收集**：检索多个公开数据源
3. **分析验证**：交叉验证信息准确性
4. **发现新线索**：从结果中提取新的搜索关键词
5. **扩大搜索**：基于新线索扩展搜索范围
6. **循环迭代**：重复上述过程直到满足调研目标

### 种子扩散策略

- **企业种子**：公司名 → 关联企业 → 投资关系 → 供应链关系
- **人员种子**：关键人名 → 任职历史 → 关联企业 → 投资版图
- **产品种子**：产品名 → 竞品对比 → 市场份额 → 技术路线

## 操作步骤

### 步骤1：需求解析与种子识别

**智能体处理：**
- 分析用户调研需求，明确调研目标
- 识别初始种子信息（目标企业、人员、产品等）
- 确定调研范围和深度要求
- 制定调研策略和数据源优先级

**输出：**
- 调研任务清单
- 初始种子列表
- 数据源优先级排序

### 步骤2：多源信息采集

**调用 `scripts/collect-data.js`：**
- 根据种子信息自动检索多个公开数据源
- 支持的数据源类型：
  - 企业工商信息（天眼查、企查查等公开数据）
  - 新闻资讯（搜索引擎、新闻聚合）
  - 社交媒体（LinkedIn、Twitter等公开信息）
  - 行业报告（公开的行业研究报告）
  - 专利和知识产权（公开专利数据库）

**参数：**
```javascript
{
  seeds: ['公司名', '人名', '产品名'],
  dataSources: ['enterprise', 'news', 'social', 'patent'],
  depth: 1-3, // 迭代深度
  cacheDir: '/tmp/osint-cache'
}
```

### 步骤3：数据清洗与关联分析

**调用 `scripts/analyze-relations.js`：**
- 清洗和标准化采集的原始数据
- 构建企业关联关系图谱
- 识别关键节点和关系路径
- 提取有价值的洞察信息

**输出：**
- 结构化数据（JSON格式）
- 关系图谱数据
- 关键发现摘要

### 步骤4：报告生成与可视化

**智能体处理：**
- 基于分析结果生成调研报告
- 创建可视化图表（关系图谱、时间线等）
- 整理关键发现和建议

**输出格式：**
- Markdown格式的调研报告
- 关系图谱可视化（HTML/SVG）
- 原始数据导出（JSON/CSV）
